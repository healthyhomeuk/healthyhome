Core::StatusCode LinuxI2C::Driver::open ( int  deviceId)

Opens an I/O stream to an I2C device.

This function is essential so that the I2C operations can be performed. It opens a I/O stream to a Linux I2C device file of the name /dev/i2c-X, where X represents the deviceId.
