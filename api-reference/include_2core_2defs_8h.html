◆ StatusCode

enum Core::StatusCode

Operations status codes type.

This enumerator helps keeping consistency in operation statuses throughout the whole project. Any method that performs an operation a returns a status code, should use this enumerator type.

Usage example:

Core::StatusCode computation(int a, b) {
// perform super long computation
return Core::SUCCESS;
}
Enumerator
SUCCESS 

returned if the operation was successful.

E_BUSY 

returned if the operation was not performed due to the given resource being busy.

E_CONFLICT 

returned if the operation was not performed due to a conflict with the given resource.

E_OUT_OF_BOUNDS 

returned if the provided parameters are out of bounds.

E_PARAMS 

returned if the given parameters are invalid

E_NOT_FOUND 

returned if the requested resource was not found

E_GENERIC 

returned for any other error.
